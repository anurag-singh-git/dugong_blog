## New post

**Dispatch Nº 79 — "The order you split across vendors by hand"**
`how-to-automatically-split-and-route-shopify-orders-to-multiple-suppliers.html` · by Priya Singh · 2026-07-07 · ~2,456 words

Chosen because "how do I automatically split an order by vendor / send each supplier only their
part / email a purchase order to my dropshipper" is one of the most-repeated, still-unsolved requests
for any store that fulfils from more than one source, and no existing dispatch covered it. Forum and
app-ecosystem validation is strong: a dense cluster of routing and dropship apps exists precisely
because the native platform cannot do it (SplitOrder, OrderSplit Pro, Order Fulfillment Guru, Duoplane,
Vendor Notify, Order Trigger, Smart Notifications, Order Desk), and Shopify's own changelog and help
pages confirm the native split is by *location*, not by *supplier*.

The native gap is real and specific, and the post credits the parts Shopify does. Shopify genuinely
splits fulfilment across your own stocking locations when one is short, and offers split shipping at
checkout, which covers multi-warehouse stores. But that split is by location and inventory priority,
not by which supplier makes an item; there is no rules engine to route a line to a vendor by
vendor/type/SKU/weight; the product **vendor field is only a label** that notifies no one and generates
no purchase order; and there is **no native action** that sends each supplier a document with just their
line items in their format. Shopify Flow can tag an order and (on higher plans) move line items between
locations, but has no action that groups lines by supplier, builds a per-vendor PO, sends it, and then
fulfils only their lines when tracking returns. The post frames the differentiator: not one more routing
rules-table, but a read + classification into supplier groups + a document build per vendor + an
automatic send + a chain of partial fulfilments that close as each supplier ships, with judgement (a
silent supplier, a backordered line, a delivery-fragmenting split) escalated to a human.

The heading screams the use case ("the order you split across vendors by hand"); the H1 carries the
high-intent search phrasing ("How to automatically split and route Shopify orders to multiple
suppliers"). Target phrases: "Shopify split order by vendor," "Shopify route order to supplier,"
"Shopify multi-vendor fulfillment," "Shopify split order multiple suppliers," "Shopify dropship order
routing," "Shopify send purchase order to supplier," "Shopify vendor split," "Shopify split
fulfillment," and "Shopify 3PL order routing."
