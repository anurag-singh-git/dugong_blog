## New post

**Dispatch Nº 85 — "The gift message that never reaches the box"**
`how-to-add-a-gift-message-and-gift-wrap-at-checkout-on-shopify.html` · by Priya Singh · 2026-07-18 · ~2,090 words

Chosen because gifting at checkout is one of the steadiest, still-unsolved complaints in the Shopify
community, and it has three distinct failure modes that stack into one bad outcome. Forum validation is
strong and verbatim: recurring threads titled "How can I add a gift note to my packing slip?", "Add a
gift message option at checkout", and "How do I hide prices on the slip for a gift?" run across the
Shopify community and help centers. The pain is also seasonally sharp for a timeliness angle: the fourth
quarter turns most carts into gifts, so this is a high-intent, in-the-moment search for a large share of
merchants. No existing dispatch covered it. The closest posts (free-gift-with-purchase, order tagging,
combine-orders, send-invoices) are cart-value gifting, generic tagging, order merging, and B2B invoices,
not gift messages, gift wrap, or a price-free gift packing slip. It was on the carry-forward candidate
list ("gift-message/gift-wrap") from the 07-16 SEO pass.

The native gap is real, specific, and credited. Shopify does give the shopper one native text field, the
cart order note, which can be enabled and renamed, and B2B/Plus stores have some checkout options. But
(1) the order note is a single freeform field for the whole order, unlabeled, off by default in many
themes, not per-item, with no validation and no signal that the order is a gift; (2) gift wrap at
checkout is native only for Shopify Plus on Checkout Extensibility, and even then a bare toggle rather
than a fee, a line item, and a bench instruction; (3) the packing slip is one shared template that prints
prices, with no native switch to hide totals for a gift, so the recipient can open a box showing what the
sender paid; and (4) getting the message onto the slip needs Liquid edits and a manual template habit that
breaks under load and when a warehouse app takes over fulfillment. The post frames the differentiator:
capture the gift toggle, wrap choice, and message as order attributes; validate and clean the message;
attach wrap as a real line item or fee; tag the order as a gift; generate a gift packing slip with the
message in and the prices out, routed to the bench; and escalate the judgement calls (offensive or
oversized messages, multi-address gifts, split shipments).

The pain-point heading screams the use case ("the gift message that never reaches the box"); the H1
carries the high-intent search phrasing ("How to add a gift message and gift wrap option at checkout on
Shopify"). Target phrases: "Shopify gift message," "Shopify gift wrap at checkout," "Shopify gift note
packing slip," "Shopify gift receipt hide prices," "add gift message option Shopify," "Shopify gift option
not working," "Shopify order notes gift," and "Shopify gifting checkout."
