## New post

**Dispatch Nº 84 — "The buy-more-save-more discount Shopify won't tier"**
`how-to-set-up-tiered-volume-discounts-on-shopify.html` · by Priya Singh · 2026-07-16 · ~2,140 words

Chosen because tiered / volume discounts (buy 3 save 10, buy 6 save 20, buy 12 save 30) are among
the most-repeated, still-unsolved complaints in the Shopify community. Forum validation is strong and
verbatim: threads titled "Tiered discounts based on variant quantity," "Tiered buy X, get Y discount?,"
and "how do I do buy more save more without an app" recur across the community. The pain is also
freshly urgent for a GEO/timeliness angle: Shopify Scripts (the way Plus merchants faked tiered cart
logic for years) stopped executing on June 30, 2026, so any store that relied on Scripts now has a
broken checkout, which makes this a high-intent, in-the-moment search. No existing dispatch covered it:
the closest posts (bulk-edit-product-prices, schedule-a-sale, free-gift-with-purchase,
set-a-minimum-order-amount) are manual repricing, sale timing, cart-value gifting, and order minimums,
not a real buy-more-save-more quantity ladder.

The native gap is real and specific, and the post credits what Shopify does. Native automatic discounts
handle the simple single-tier case cleanly ("buy 3 get 10% off"), and can gate on collection, minimum
quantity, or minimum spend. But (1) each discount holds one condition, so multiple tiers cannot be
chained into one storefront offer; (2) faking it with three separate discounts backfires, because a big
cart qualifies for all of them and Shopify does not pick the best tier and suppress the rest; (3) there
is a hard cap of 25 active automatic discounts; (4) the only native feature that tiers correctly is B2B
quantity pricing, which applies only to customers assigned to a company profile, not the open storefront
or tagged retail customers; and (5) Shopify Scripts, the old escape hatch, stopped running June 30, 2026.
The post frames the differentiator: read a plain-language tier table, resolve the single highest tier a
cart earned on every cart change, apply only that tier, guard stacking against a margin floor, apply the
right ladder to the right audience (wholesale vs retail), nudge the shopper toward the next rung, and
escalate the judgement calls (floor crossings, code clashes, overlapping promotions).

The pain-point heading screams the use case ("the buy-more-save-more discount Shopify won't tier"); the
H1 carries the high-intent search phrasing ("How to set up tiered volume discounts (buy more, save more)
on Shopify"). Target phrases: "Shopify tiered discount," "Shopify volume discount," "Shopify buy more
save more," "Shopify quantity break discount," "Shopify tiered pricing not working," "Shopify multiple
discount tiers," "bulk discount Shopify," and "Shopify quantity discount native."
